# CareClaw Workflow Skill

Use this skill when handling CareClaw healthcare consultation workflows.

Steps:

1. Identify the current consultation stage.
2. Select the next agent or tool based on workflow state.
3. Produce structured output.
4. Preserve doctor authority for diagnosis, prescription, and final patient delivery.
5. If red flags are present, route to urgent doctor review.
6. If payment is missing, create a payment handoff before doctor chat.
